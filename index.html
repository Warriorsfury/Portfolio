This is my project on Space Station 13.

An old roleplaying game. Inside is pretty much a "cheat sheet" kind of wiki for various things on the main server I played on. Mainly used by either myself, or friends. Nothing major about it, and infact it's one of the first few things i've done in web development.

It also contains the first thing i've fiddled with in Javascript/Jquery. One of the more difficult languages i've been learning. You will find various "sections" inside about chemistry, security, engineering, and laws. It is quite likely that i'll go in and touch it up a fair bit in the future.

No inspiration was used for the main development of the site. The only programs used to create it were purely Sublime Text. Nothing like Bootstrap or Wordpress was used either. Purely built from the ground up. The main reason as to why it was made was to help myself learn the different chemistry mixes to create them effectively and quickly. Soon after completing a baseline of what I wanted, I showed friends who also played the game, and they took a liking to it too and begun to use it to help themselves as well.
